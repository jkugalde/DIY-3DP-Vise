# DIY 3DP Vise

[![CC BY 4.0][cc-by-sa-shield]][cc-by-sa]

It is a small vise

<img src="/img/isovise.png" width="600">

## Features

- Cheap
- 115 mm opening
- Removable handle
- Fixed size, not affected by the movement of the jaw
- Rubber surface jaw
